# ToDo-App 

### Technologies Used
- ReactJS
- NextJS
- Tailwind CSS
- Spring Boot
- Java Programming
- MySQL DataBase

### How to Run ?
- Start MySQL Server using XAMPP
- Start Java Backend using IntelliJ Idea IDE
- `cd client/`
- `npm run dev`
- Goto http://localhost:3000

### Some Information
- Java Backend Port: 8081
- Frontend Port: 3000
